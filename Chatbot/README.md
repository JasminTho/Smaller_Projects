# Chatbot GUI App

## Assignment Description
This project demonstrates a simple chatbot using a decision tree classifier and a bag-of-words approach. The user can interact with the chatbot through a GUI built with Tkinter. The task was to train a model on predefined patterns and tags, and respond with randomly selected answers from the corresponding category.

## Installation
The program runs on all common operating systems. To start the program, install Python 3.10 or higher and the following modules:
- pandas
- Pillow
- scikit-learn
- tkinter

The dataset must be in CSV format and named ChatbotTraining.csv. Additionally, an image file sending.png is required for the GUI button.
