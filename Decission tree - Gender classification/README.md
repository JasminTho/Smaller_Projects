# Gender Classification App 
## AssignmentDescription 
This program demonstrates a simple machine learning workflow using a decision tree classifier. The task was to load a dataset, train a model, and predict gender based on features, with a simple GUI for user interaction. 

## Installation 
The program runs on all common operating systems. 
To start the program, install Python 3.10 or higher and the following modules:
- pandas 
- scikit-learn 
- tkinter  

The dataset must be in CSV format and named gender_classification.csv.
